# コンセプト
- パソコンなどの機械慣れしていないの私の母のような方でも、簡単に遊べて、頭の運動になるようなゲームというコンセプトをもとに、要件定義、設計、実装しました。
- どこかで見たことがあるようなゲーム化もしれませんが、温かい目でご覧いただければ幸いです。

# ゲーム概要
- 本ゲームでは、ユーザーが操作できるキャラクターが2体おり、操作したいキャラクターを適宜入れ替えて、操作し、進めていくゲームとなっています。
- 十字キー、スペースキー入力でキャラクターの操作を行い、マウス入力でキャラクターの入れ替え等を行います。

# 使用技術
- 開発言語：C++
- 開発フレームワーク：cocos2d-x(v4)
- 開発環境：VisualStudio2022
- 検証済みOS：Windows11

# 技術の選定理由
- 本開発では、ゲームエンジンの仕組みを理解しながら2Dゲームを開発することを目的としてcocos2d-xを選定しました。cocos2d-xはC++で記述されており、メモリ管理やScene/Layer/Node構造、物理演算のコールバックなどを意識しながら実装できるため、ゲームエンジンの内部構造への理解を深められると考えました。
- 他の選択肢として、Unityなどの高水準エンジンも検討しましたが、内部処理が抽象化されている部分が多く、挙動を深く理解しにくいと感じました。
- また本開発では、学習フェーズとして単にゲームを完成させるだけでなく、「なぜその挙動になるのか」を理解しながら開発することを重視しました。そのため、私たち自身で内部処理を理解し、エラー処理などを行っていく過程を通して、問題解決する力を伸ばしていける点も本開発の要件に適していました。

# 実装予定の機能
- ステージ選択画面で、ステージを選択後にステージプレビュー用のLayerを追加。そのLayer内でスターの条件なども見れるように。
- ステージの拡張
- 鍵などの新ギミックの追加
- セーブ、ロード機能の実装
